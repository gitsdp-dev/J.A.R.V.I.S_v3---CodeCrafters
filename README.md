# 🚀 What is J.A.R.V.I.S v3?

J.A.R.V.I.S v3 is the latest generation of the **J.A.R.V.I.S** project developed by **CodeCrafters**.

More than just an AI chatbot, J.A.R.V.I.S v3 is a complete **AI-powered desktop operating companion** designed to simplify multitasking, research, productivity, and computer automation.

Using natural conversations, J.A.R.V.I.S can understand your requests, interact with your operating system, analyze uploaded documents, monitor your screen, execute complex workflows, and even allow you to control your computer remotely from your smartphone.

Built with a futuristic interface inspired by advanced AI systems, J.A.R.V.I.S makes interacting with your computer feel effortless.

# 🛠 Installation and First-Time Run

## 1. Clone the Repository

```bash
git clone https://github.com/gitsdp-dev/J.A.R.V.I.S_v3---CodeCrafters.git
```

## 2. Navigate into the Project

```bash
cd J.A.R.V.I.S_v3---CodeCrafters
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
playwright install
```

## 4. Run J.A.R.V.I.S

```bash
python main.py
```

On the first launch, J.A.R.V.I.S will guide you through the initial setup process, including configuring your free Gemini API key.
Then, you can click on the Create Desktop Shortcut button to create a shortcut, which can open J.A.R.V.I.S like an application software.
